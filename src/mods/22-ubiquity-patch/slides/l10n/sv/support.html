Hjälp och Support

Den officiella dokumentationen för Cortex Linux täcker ett brett spektrum av problem du kan stöta på. Dessutom är Cortex Linux community-forum fyllda med användare och utvecklare.

På Ask Ubuntu kan du söka efter och fråga om mycket vanliga problem. Lösningar för nästan alla Cortex Linux-problem är identiska med de för Ubuntu. Du kan helt förlita dig på Ubuntus dokumentation och expertis för att lösa problem med Cortex Linux.
