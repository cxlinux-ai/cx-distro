Hilfe und Unterstützung

Die offizielle Dokumentation für CX Linux deckt eine Vielzahl von Problemen ab, auf die Sie stoßen könnten. Zusätzlich sind die CX Linux Community-Foren voll von Benutzern und Entwicklern.

Auf Ask Ubuntu können Sie nach sehr häufigen Problemen suchen und Fragen dazu stellen. Lösungen für fast alle CX Linux-Probleme sind identisch mit denen für Ubuntu. Sie können sich vollständig auf die Dokumentation und das Fachwissen von Ubuntu verlassen, um Probleme mit CX Linux zu lösen.
